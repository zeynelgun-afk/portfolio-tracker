# 📋 Strateji Açıklaması

## Genel Yaklaşım

Multi-sector value + momentum blend. Enerji sektörüne ağırlık verilmiş, altın/bakır ile komodite çeşitlendirmesi yapılmış, MO ile defansif gelir bileşeni eklenmiştir.

## Pozisyon Seçim Kriterleri

1. **Değerleme**: P/E sektör ortalamasının altında veya makul
2. **Momentum**: Pozitif fiyat trendi (50 DMA üzeri)
3. **Katalitör**: Yaklaşan earnings, sektörel rüzgar, makro destek
4. **Risk/Getiri**: Minimum 2:1 risk/getiri oranı

## Risk Yönetimi

- **Tek pozisyon max**: %25
- **Tek sektör max**: %40
- **Minimum nakit**: %3
- **Stop-loss**: -%15 (pozisyon bazında)
- **Earnings riski**: Bilanço öncesi pozisyon küçültme değerlendirilir

## Rebalancing Kuralları

- Haftalık gözden geçirme (Pazar)
- Aylık rebalancing (gerekirse)
- Earnings öncesi/sonrası özel değerlendirme
- Makro rejim değişikliğinde taktik ayarlama

## Çıkış Kriterleri

1. Tez bozulması (fundamentaller kötüleşirse)
2. Stop-loss tetiklenmesi
3. Hedef fiyata ulaşma
4. Daha iyi fırsat ortaya çıkması (opportunity cost)
5. Sektörel / makro risk artışı
